IoT-Based Patient Assistance System

Innovative healthcare solution for automated medication management

Problem Statement

Medication management is a significant challenge for elderly patients. Missed doses, incorrect timings, and over-reliance on caregivers can lead to severe health complications. Traditional methods often fail to provide a comprehensive solution, creating a need for an automated, reliable system.

Key Features

  • ✔ Automated pill dispensing using a rotary carousel mechanism.
  • ✔ Real-time alerts via a connected mobile app.
  • ✔ LCD display for dosage history and upcoming reminders.
  • ✔ User-friendly push-button interface for elderly patients.
  • ✔ Remote monitoring for caregivers and family members.

Design & Implementation

The system uses an ESP32 microcontroller for seamless connectivity. A stepper motor drives a rotary pill dispenser, controlled via a push-button interface. The LCD display shows key information, and a mobile app provides real-time notifications and tracking.
